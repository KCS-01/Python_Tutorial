# 리스트

여러 데이터를 모아둘 경우에 필요

여러 데이터를 저장하기 위한 자료형

## 생성

```py
animals = ["사자", "호랑이", "고양이", ... , "강아지"]
```

## 접근

```py
animals[0]
animals[1]
animals[2]
animals[3]
```

## 추가

```py
animals.append("데이터")
```

## 삭제

```py
del animals[1]
```

## 슬라이싱

```py
newList = animals[0:2]
```

## 길이

```py
len(animals)
```

## 정렬

```py
animals.sort()
```
